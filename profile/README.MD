# STHO — AI-Powered Live Entertainment Infrastructure

STHO is building the infrastructure layer powering the next generation of live entertainment.  
Our platform connects artists and venues through AI-driven workflows, intelligent matching, and automated booking pipelines.

We focus on transforming a highly fragmented and manual industry into a scalable, data-driven ecosystem.

---

## Mission

Our mission is to become the operating system for live entertainment.

We enable:

- Artists to scale their careers
- Venues to optimize booking and programming
- The industry to operate with modern infrastructure and AI automation

---

## Platform Overview

STHO provides:

- AI-powered artist and venue profiles
- Intelligent booking workflows
- Equipment and show requirement management
- Automated operational pipelines
- Scalable infrastructure designed for global deployment

The platform is designed from day one for scalability, reliability, and automation.

---

## Technical Architecture

STHO uses a modern, production-grade architecture built around containerized services and automated deployment pipelines.

### Core stack

**Frontend**
- Next.js (App Router)
- TypeScript
- TailwindCSS
- shadcn/ui

**Backend**
- Laravel (PHP 8+)
- REST API architecture
- JWT authentication
- Queue workers and async processing

**Infrastructure**
- Docker containerization
- Traefik reverse proxy
- GitHub Actions CI/CD pipelines
- GitHub Container Registry
- Multi-environment deployment (local, staging, production)

**Database**
- PostgreSQL

---

## Infrastructure Principles

Our infrastructure follows strict engineering standards:

- Immutable deployments via CI/CD

- Fully containerized services
